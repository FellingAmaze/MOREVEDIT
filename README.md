- [ ] Вход в приложение выполнен

- [ ] Подтверждение разрешений
- [ ] Отображение данных

- [ ] Корректный подсчёт шагов

- [ ] Выбор Health Connect
- [ ] Выбор разрешений
- [ ] Синхронизация данных

- [ ] Отображение данных

- [ ] Отображение пульса

### 2. Выгрузка активности
- [ ] Данные отображаются на экране «Активность»

- [ ] История за день/неделю/месяц

### 3. Учёт активности
- [ ] Подсчёт шагов, калорий, дистанции
- [ ] Расчёт баллов здоровья

### 4. Обработка ошибок
- [ ] Ошибки при отсутствии интернета
- [ ] Отказ в разрешениях

## Тест-кейсы для функционала авторизации/регистрации

### TC-AUTH-001 — Регистрация через Email
**Предусловие:** Пользователь не авторизован

| Шаг | Действие | Ожидаемый результат |
|-----|----------|---------------------|
| 1 | Открыть приложение | Экран входа |
| 2 | Ввести email | Email принят |
| 3 | Принять соглашение | Чекбокс отмечен |
| 4 | Нажать «Продолжить» | Код отправлен на email |
| 5 | Ввести код из письма | Код принят, ввод никнейма |
| 6 | Ввести никнейм | Никнейм принят |
| 7 | Ввести параметры (дата рождения, пол, вес, рост) | Параметры приняты |
| 8 | Нажать «Готово» | Вход выполнен |

### TC-AUTH-002 — Регистрация с существующим email
- Ввести уже зарегистрированный email → сообщение «Пользователь с таким email уже существует»

### TC-AUTH-003 — Невалидный email
- email без @ → ошибка
- пустая строка → ошибка
- пробелы → ошибка

### TC-AUTH-004 — Вход через VK
- Нажать «Через VK» → авторизация VK → возврат в приложение → пользователь авторизован

### TC-AUTH-005 — Вход через Google
- Нажать «Через Google» → выбор аккаунта → подтверждение → пользователь авторизован

### TC-AUTH-006 — Вход через Apple ID
- Нажать «Через Apple ID» → подтверждение → пользователь авторизован

### TC-AUTH-007 — Вход существующего пользователя
- Ввести email → код на email → ввести код → вход выполнен

### TC-AUTH-008 — Неверный код подтверждения
- Неверный код → сообщение об ошибке
- Просроченный код → запросить новый
- Пустой код → кнопка неактивна

### TC-AUTH-009 — Отказ от соглашения
- Не отметить чекбокс → кнопка «Продолжить» неактивна

### TC-AUTH-010 — Выход из аккаунта
- Профиль → Настройки → Выйти → подтвердить → экран входа
